   - Exportar base de datos: if0_39762769_flota_db
   - Guardar como: flota_backup_[fecha].sql

2. LIMPIAR SERVIDOR REMOTO:

   - Eliminar TODOS los archivos y carpetas

3. SUBIR ARCHIVOS:

   - Verificar que se suban correctamente

4. RESTAURAR BASE DE DATOS:
   - Importar el backup en phpMyAdmin
   - Verificar que se restaure correctamente

5. PROBAR:

   - Probar registro de conductor
   - Verificar que no haya errores HTTP 500

6. OPTIMIZAR:
